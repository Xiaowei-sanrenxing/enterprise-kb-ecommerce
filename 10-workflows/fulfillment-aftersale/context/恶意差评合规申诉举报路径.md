# 恶意 / 同行差评 合规申诉举报路径（★守住"不删评"红线后，给正向出路）

> 补盲区：[[退货归因与评价管理]] 已明确「不得删差评/不得利诱改评」红线，但只能答"按平台规则处理"。
> 本流程专管**疑似恶意、同行攻击、敲诈勒索类差评**——agent 不删评、不对线，但要给出**合规的申诉/举报正向通道**。

## 第一判断：是哪一类差评（决定走哪条路）
| 类型 | 信号 | 走向 |
|---|---|---|
| 真实负评 | 买家真买了、真有体验问题 | 走 [[退货归因与评价管理]]：真诚联系解决 + 回流改进，**不申诉** |
| **疑似恶意/同行** | 未购买却评、批量小号、内容与商品无关、纯人身攻击、明显抹黑 | 走本流程，**走平台申诉** |
| **敲诈勒索** | "不退款/不赔钱就继续差评""给我返现就删评" | 走本流程，**截图固证 + 平台举报 + 必要时报警/法务** |

## 处置四步（agent 协助整理材料，不自行删评/对线）
1. **取证固定**：第一时间截图/录屏差评内容（评价时间、买家账号、订单号或"无订单"证据、聊天威胁记录），不删不公开回怼。
2. **判定理由对号入座**：按平台恶意评价申诉常见受理理由归类——
   - 未发生真实交易（无对应订单）
   - 内容与商品/服务无关、纯广告或人身攻击
   - 同一买家/小号批量恶意评价
   - 含敲诈勒索、违法违规信息
   - 泄露他人隐私或含不实诋毁
3. **走平台官方申诉/举报通道**（由运营/售后主管提交，agent 整理材料）：
   - 商家后台「评价管理 / 评价申诉」入口提交申诉，附第 1 步证据
   - 平台「不正当竞争 / 恶意评价举报」通道（针对同行水军）
   - 敲诈勒索类：保留证据 → 平台举报 + 必要时报警，转法务评估
4. **正向回应**：申诉期间可在差评下做**克制、就事论事的公开回复**（说明已联系/已申诉、不攻击买家），避免二次发酵。

## 🔴 agent 绝不做
- 不自行删差评、不找平台/第三方"刷掉"差评（属 [[退货归因与评价管理]] 绝对红线）
- 不利诱/胁迫买家改评或撤评（"返现删评"既违规又可能正中敲诈圈套）
- 不在评论区和买家/同行对线互撕、不捏造反击内容
- 不擅自把差评定性为"恶意"对外宣称——定性由平台申诉裁定，对外措辞从严

## agent 标准回应（老板问"这条恶意差评能不能删"时）
"删差评是违规红线，我不能删，也不能找人刷掉。但这条像同行恶意/敲诈，可以走正向通道：① 我先把证据截图固证（时间、账号、有无真实订单、威胁记录）；② 按'未真实交易/与商品无关/敲诈勒索'对号申诉，由〔售后主管〕在评价管理后台提交平台申诉；③ 如属同行水军走不正当竞争举报，敲诈勒索还要留证报警转法务。材料我已帮你整理好，等你确认提交。"

## 需人工确认 / 上报
- 申诉提交、对外公开回复定稿 → 〔售后主管〕确认
- 敲诈勒索、批量水军、涉法律 → 上报〔合规 DRI〕+ 法务
