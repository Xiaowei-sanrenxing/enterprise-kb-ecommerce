# 内容与流量 SOP · 人可读总览

> 给人看的全流程。Agent 执行请走 [`_task-card.md`](_task-card.md)。

## 一句话流程
搭承接（详情页/直播按三屏三段）→ 过合规自检 → 查承接力达标 → 小预算测款 → 三关过再放量 → 守 ROI/库存 → 沉淀爆款结构。

## 几条铁律
1. **先承接后放量**：转化承接力不达标就加投流 = 放大亏损。
2. **卖点说好处不说参数**：「出门一天不断电」胜过「2000mAh」。
3. **测款再放量**：新品小预算测点击/转化/ROI，三关过才加钱。
4. **ROI 守线**：持续低于盈亏平衡点就止损，不为冲量硬亏。
5. **合规红线**：禁极限词/虚假功效/虚构原价，评价脱敏、禁刷单控评。

# 流量结构与投放红线（★花钱买量前必读）

> 核心：**先有转化承接力，再放量；ROI<1 持续投 = 烧钱给平台。**
> 流量分自然+付费两条腿，付费是放大器——放大的是转化效率，承接力差时放大的是亏损。

## 流量结构（三条腿）
| 来源 | 特点 | 打法 |
|---|---|---|
| 自然流量 | 免费、靠权重/排名 | 养：销量+好评+DSR 推坑产，长期地基 |
| 付费投流 | 花钱、可控、即时 | 直通车/引力魔方/千川等，按 ROI 调 |
| 内容/直播 | 短视频/直播间种草 | 内容种草 + 直播逼单，导流回详情页/链接 |

## 投放红线（agent 投放前自检）
1. **承接力优先**：详情页转化率/直播间转化没达健康线（如 <行业基准）→ 先优化承接，别加投流。负毛利或转化太差时放量=放大亏损。
2. **小预算测款再放量**：新品先小预算测点击率/转化率/ROI，跑出正反馈再加预算，不一上来梭哈。
3. **ROI 守线**：算清盈亏平衡 ROI（含毛利/退货），持续 <平衡点要止损或优化，不为冲量硬亏。
4. **断货不投**：库存不足时降速/暂停投放（见选品供应链），避免售罄掉权重。

## 测款节奏
```
小预算测 → 看点击率(够不够吸引) → 看转化率(承接行不行) → 看 ROI(赚不赚)
三关都过 → 逐步加预算放量；任一关挂 → 优化对应环节，不盲目加钱
```

## 直播/内容要点
- 直播间也是「三屏逻辑」：开场抓人、中段说服、逼单促成。
- 投流千川等同理：先测内容素材，跑出爆款素材再放量。
- 一切宣传遵守合规红线（禁极限词/虚假功效/虚构价格）。

## 数据
投放后台数据、ROI、成本属 🟡 内部，不对外。

# 详情页黄金三屏（★转化主战场）

> 核心：**首屏抓人（3 秒留住）→ 中屏说服（解决疑虑）→ 尾屏逼单（临门一脚）。**
> 转化率低，80% 是详情页结构没搭对，不是流量不够。

## 三屏结构

### 首屏 · 抓人（决定跳出率）
- 一句话讲清「卖给谁 + 解决什么问题 + 凭什么是你」。
- 主图突出核心卖点/使用场景，不堆字、不放无关美图。
- 3 秒内让访客判断「跟我有没有关」，没说清就跳走。

### 中屏 · 说服（决定加购）
- 卖点场景化：把参数翻译成「对你的好处」（不是「容量2000mAh」而是「出门一天不断电」）。
- 打消疑虑：材质/工艺/实测/对比/资质（事实可验证，禁极限词，见合规默认）。
- 信任背书：真实评价/销量/使用前后对比（脱敏，禁刷单控评）。

### 尾屏 · 逼单（决定下单）
- 临门一脚：限时优惠/赠品/无理由退货承诺/物流时效。
- 降低决策成本：七天无理由、运费险、售后保障说清楚。
- 不虚构原价/不先涨后降（价格合规红线）。

## 自检清单（发布前过）
- [ ] 首屏 3 秒能看懂卖给谁、解决什么
- [ ] 卖点是"好处"不是"参数"
- [ ] 无极限词/无虚假功效/无虚构原价
- [ ] 评价/案例已脱敏，无刷单痕迹
- [ ] 尾屏有明确下单理由 + 售后保障

## 用法
详情页是承接所有流量的落点。投放/直播引来的流量，最终都靠它转化，先把它搭对再加 [[流量结构与投放红线]] 的量。

# 权限分级 · 四级

> 企业库和个人库最硬的区别。没有权限治理 = AI 资料垃圾场 + 隐私事故。

| 级别 | 含义 | 典型资产 | agent 行为 |
|---|---|---|---|
| 🟢 公开 | 可对外发布 | 对外物料、方法论、介绍 | 可写入对外产物 |
| 🟡 内部 | 团队内用 | 话术、SOP、运营数据 | 内部可用,不对外 |
| 🔴 受限 | 隐私/敏感 | 客户对话、通话记录、手机号、名单 | **绝不写入对外产物,只内部分析** |
| ⛔ 禁止入库 | 不该进库 | 证件、银行信息、未授权第三方资料 | 拒绝入库 |

## agent 红线
1. 🔴 受限资产 → 输出时必须脱敏/打码,绝不原文外发
2. 超标准的报价/优惠/对外承诺 → 必须人工确认
3. 不确定权限 → 默认按更高一级处理,问用户

## 默认权限分级（通用四级 + 行业补充）

|---|---|---|
| 客户聊天原文 / 订单信息 / 收货地址 / 手机号 / 微信号 | 🔴 受限 | 绝不写入任何对外产物，绝不进飞书，引用必脱敏 |

## 行业红线（agent 触碰即停，转人工确认）

### 1. 虚假宣传 / 极限词
- 禁用「最」「第一」「国家级」「顶级」「绝无仅有」「100%」等绝对化、极限词（《广告法》明令禁止）。
- 不得编造功效、夸大效果、虚构成分/产地/资质；保健/医疗类不得宣称疗效。
- 不得伪造「销量/好评/获奖/权威认证」等背书。
- agent 写文案/详情页一律走「事实可验证」原则，拿不准的形容词不写。

### 2. 价格欺诈
- 禁止虚构原价/划线价（划线价必须是近期真实成交价）。
- 禁止「先涨后降」假打折、虚假满减、误导性赠品。
- 标价与实付一致，不得有隐藏费用。

### 3. 七天无理由退货
- 法定支持七天无理由（生鲜/定制/拆封贴身用品等法定除外类目除外，且须事先明示）。
- 不得擅自拒绝合理无理由退货、不得设置不合理退货门槛。
- 售后 agent 默认按「支持七天无理由」处理，例外类目须有明示依据。

### 4. 刷单 / 虚假交易（绝对红线）
- 严禁刷单、刷好评、删差评、虚构物流、组织炒信。
- 严禁诱导/胁迫好评（如「好评返现」属违规）、严禁恶意操纵评价。
- agent 不得生成任何刷单/控评相关内容或话术，遇到此类需求直接拒绝并提示风险。

### 5. 消费者权益
- 不得泄露、买卖客户个人信息（订单/地址/手机号）。
- 不得在客户未同意下骚扰式营销（短信/电话轰炸）。
- 退换货、发票、质量问题须依法履约，不得推诿。

### 6. 平台规则
- 遵守所在平台（淘系/抖音/拼多多/京东等）的类目资质、发布、营销、物流时效规则。
- 不得跨平台引流违规、不得绕过平台交易（私下转账规避保障）。
- 类目准入资质（如食品经营许可/品牌授权）须齐全方可经营。

## 权限分级处置原则
- agent 默认按「最严」处理：拿不准级别 → 当受限处理 → 问负责人。
- 涉及客户隐私、超授权报价/优惠、价格/宣传合规判断 → 须人工确认，不自动执行。
- 任何对外产物（详情页/朋友圈/案例/物料）发布前过一遍本红线清单。
