# 电商行业 黑话词典 · 让 agent 看懂内部术语

> 行业通用术语已预填。店铺专属术语 / 人物 / 产品 / 自定义指标请客户补在文末。

## 核心指标（行业通用，agent 看到数字先认得是什么）
- **GMV**：成交总额（下单金额，含未支付/退款），衡量盘子大小，≠ 实际到手收入。
- **销售额/支付GMV**：实际支付成交额，比 GMV 更接近真金白银。
- **UV / PV**：UV=独立访客数，PV=页面浏览量；UV 是有多少人来，PV 是看了多少次。
- **转化率（CVR）**：成交订单数 ÷ 访客数，电商最核心的健康指标。
- **客单价（AOV）**：平均每笔订单金额 = 销售额 ÷ 订单数。
- **UV价值**：每个访客平均贡献的销售额 = 销售额 ÷ UV，衡量流量质量。
- **ROI / ROAS**：投产比，付费推广带来的销售额 ÷ 花掉的广告费，<1 = 亏本投流。
- **CPC / CPM / CPA**：点击成本 / 千次曝光成本 / 单次转化成本，投放三大计价口径。
- **加购率 / 收藏率**：加入购物车、收藏的占比，衡量种草强度（直接影响后续转化）。
- **退货率 / 退款率**：退货退款占比，电商利润杀手，售后工作流核心监控项。
- **复购率**：一定周期内重复购买的客户占比，私域/会员体系的核心 KPI。
- **DSR / 店铺评分**：描述/服务/物流三项动态评分，影响流量分配和店铺权重。
- **好评率 / 差评率**：评价质量，直接影响详情页转化与平台权重。

## 运营术语（行业通用）
- **SKU / SPU**：SKU=最小库存单位（颜色+尺码这种具体一款），SPU=标准产品单元（一个商品款）。
- **链接 / 宝贝**：平台上的一个商品页面（商品详情页）。
- **爆款 / 链接打爆**：把一个 SKU 做成高销量主力款，靠它带动整店流量。
- **测款**：用小预算投流测试哪个款有潜力，再决定主推谁。
- **黑五类 / 极限词**：违规高风险类目 / 「最」「第一」等被禁的宣传词（虚假宣传红线）。
- **坑产**：单个商品坑位的产出（销售额），平台用它排自然排名。
- **流量入口**：搜索 / 推荐 / 直播 / 短视频 / 活动等访客来源渠道。
- **付费投流 / 自然流量**：花钱买的流量（直通车/引力魔方/千川）vs 平台免费给的。
- **直通车 / 引力魔方 / 万相台 / 千川**：各平台主流付费推广工具（淘系/抖系）。
- **私域**：可反复免费触达的自有流量池（微信/社群/会员），区别于平台公域流量。
- **达人 / 带货 / 坑位费 + 佣金**：找网红卖货，坑位费=出场费，佣金=按成交抽成。
- **铺货 / 一件代发**：不囤货，下单后由供应商直接发货给买家。
- **七天无理由**：法定无理由退货规则（售后工作流红线，见合规默认）。

## 本体系关键概念（跨工作流复用）
- **工作流**：一件能整个交给 agent 干完的事（选品/内容/销转/履约/复购），是本库组织主轴。
- **任务卡 6 要素**：目标/上下文/指令/工具/权限/验收，每条工作流一张。
- **选品三维模型**：需求/竞争/利润三轴打分定生死（见选品工作流）。
- **详情页黄金三屏**：首屏抓人/中屏说服/尾屏逼单，决定转化（见内容工作流）。
- **客户分级**：销转第一步，按意向/客单/类型快速判级决定打法（见销转工作流）。
- **退货归因闭环**：每笔退货记原因 → 回流选品/详情页/质检（见售后工作流）。
- **会员分层 RFM**：按最近购买/频次/金额分层运营（见复购工作流）。
- **受限/脱敏**：客户聊天/订单/手机号受限，对外引用必脱敏。
- **双区**：草稿区/定版区/归档区，定版才可对外、agent 只用定版交付。
